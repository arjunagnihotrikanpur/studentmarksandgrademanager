#STUDENT MARKS & GRADE MANAGER

##STUDENT MARKS & GRADE MANAGER##
###A Command-Line Interface tool used to store/calculate/display the report card of students by entering their marks
_Features_:

1. CLI Interface: for ease of access
2. Light Storage: JSON Storage for light system
3. Modular Code: Code Split into modules for scalability and debugging
4. Formatted Report Card
5. Search Student functionality to search for old students
6. Error Handling Implemented

_Tools & Technologies: _

1. Python 3
2. JSON

_Steps To Run: _
Run the python command in the program directory

```
python main.py
```

_Instructions for Testing: _

1. Run the program
2. Choose option 1 and store data of multiple students
3. Choose option 2 to look up stored data
