##PROBLEM STATEMENT
Schools and colleges need a simple system to manage student marks, compute grades, and generate report cards. Manual calculation is time-consuming and prone to errors.
This project builds a simple Python-based application that automates mark entry, grade calculation, and report generation.

##SCOPE OF PROJECT

- To provide a simple system for entering, managing, and processing student marks
- To automate the calculation of total marks, percentage, and grade
- To allow easy retrieval and display of student academic records
- To store student data in a structured, persistent format (JSON)
- To generate a readable report card for each student

##TARGET USERS

- School/college teachers who need to manage marks quickly
- Coaching centers handling multiple student assessments
- Students who want to keep track of their own performance
- Small institutions without a heavy management system

##High-Level Features

- Add student details and subject-wise marks
- Validate marks (must be between 0–100)
- Automatically compute total, percentage, and grade of each subject (between A,B,C,D,F)
- View individual student records
- List all students with grades
- Save student data to storage (JSON)
- Load previously saved data
- Generate formatted report cards
